Online Users for User Profiles Made Easy.

== Description ==
[Online Users for User Profiles Made Easy](https://goo.gl/YXqUgi) is a free addon provided by UPME  for displaying online/offline status of the users in profile and member list.

Please note that this is an addon for  [User Profiles Made Easy](https://goo.gl/w716O3) plugin and you need at least UPME version 2.2.03 to use it.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/upme-online-users` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the docs at https://goo.gl/YXqUgi to use online users.
